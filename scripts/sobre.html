Sobre o Projeto

Este projeto tem como objetivo criar uma aplicação web que permita aos usuários gerar imagens com o auxílio de inteligência artificial.

A aplicação será construída de forma incremental, acompanhando nosso aprendizado em desenvolvimento web.

  • HTML para estruturação inicial
  • CSS para estilização nas próximas etapas
  • JavaScript para interatividade
  • Integração com API de geração de imagens por IA
